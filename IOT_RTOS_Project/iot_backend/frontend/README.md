# Frontend Module

## 🎯 Mục tiêu

* Hiển thị dữ liệu sensor
* Cho phép điều khiển thiết bị

---

## 📦 Thành phần

* Dashboard (realtime)
* Analytics (history)
* Control panel

---

## 🔧 Công việc cần làm

* Gọi API từ backend
* Render UI
* Xử lý button control
* Hiển thị trạng thái

---

## 🛠 Công nghệ sử dụng

* HTML / CSS / JS
* Bootstrap
* Chart.js

---

## ⚙️ Run

* Mở trực tiếp:

```bash
http://localhost:3000
```

---

## 🧪 Test

* Check API fetch:

```js
console.log(await fetch("/api/data"))
```

* Test control:
* Click button → xem thiết bị phản hồi

---

## ⚠️ Lưu ý

* Không hardcode data
* Tách logic UI và API
